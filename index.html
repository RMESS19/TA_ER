LEÇON 01

Grandeurs Électriques de Base

Qu'est-ce que l'électricité ?

Imagine un tuyau d'eau : l'eau, c'est l'électricité. Elle coule dans les câbles. Pour qu'elle coule, il faut une pression (la tension) et un débit (le courant). La largeur du tuyau, c'est l'inverse de la résistance.

```

Les 3 grandeurs fondamentales :

U — TENSION
Force qui pousse l'électricité. Unité : Volt (V)
Analogie : la pression dans le tuyau
I — COURANT
Quantité d'électricité qui circule. Unité : Ampère (A)
Analogie : le débit d'eau
R — RÉSISTANCE
Ce qui freine l'électricité. Unité : Ohm (Ω)
Analogie : un tuyau étroit

Tableau des unités :

GrandeurSymboleUnitéSymb.
TensionUVoltV
CourantIAmpèreA
RésistanceROhmΩ
PuissancePWattW
``` Schéma : circuit simple +——[R]——+ | | [U] [R] | | +—————+ U = Batterie (12V) R = Résistance (Ω) I circule dans le sens →
```

⚡ Exemple concret — Système électrique d'un aéronef

Un moteur de volet de queue fonctionne sous 28 V (tension standard aéronautique). Le câble qui l'alimente a une résistance de 2 Ω. Le courant qui circule est calculé grâce à la loi d'Ohm (voir Leçon 2).

INFOTension standard avion militaire : 28 V (courant continu)
INFOTension secteur Suisse : 230 V (courant alternatif)
INFOBatterie voiture : 12 V — Batterie camion armée : 24 V

📌 Points clés à retenir

  • U = Tension en Volts (V) — la "pression" électrique
  • I = Courant en Ampères (A) — le "débit" d'électrons
  • R = Résistance en Ohms (Ω) — le "frein"
  • Plus R est grande → moins de courant circule (à tension égale)
  • Ces 3 grandeurs sont liées par la Loi d'Ohm
```
LEÇON 02

Loi d'Ohm — La Formule Centrale

La formule de base

La loi d'Ohm relie les 3 grandeurs. C'est la formule LA PLUS IMPORTANTE du test TA Sdt Av. Apprends-la par cœur !

```
U = R × I

On peut isoler chaque variable :

I = U ÷ R
R = U ÷ I

Le triangle magique

Pour trouver la bonne formule : cache ce que tu cherches avec ton doigt → le triangle te donne le calcul !

U
R
I

Cache U → R × I | Cache R → U ÷ I | Cache I → U ÷ R

``` Circuit avec loi d’Ohm : +—–[12V]——+ | | | [4Ω] | | +—–[I=?]——+ Données : U = 12V, R = 4Ω Cherché : I = ? I = U ÷ R = 12 ÷ 4 = 3A
```

🔧 Exemple 1 — Trouver le courant

DonnéesU = 24 V, R = 6 Ω
FormuleI = U ÷ R
CalculI = 24 ÷ 6
RésultatI = 4 A

🔧 Exemple 2 — Trouver la résistance

DonnéesU = 12 V, I = 2 A
FormuleR = U ÷ I
CalculR = 12 ÷ 2
RésultatR = 6 Ω
⚠️
Erreurs fréquentes au test :
❌ Confondre U/R et R/U — utilise toujours le triangle.
❌ Oublier les unités dans la réponse (toujours écrire A, V ou Ω).
❌ Diviser dans le mauvais sens : I = U÷R et NON R÷U.

📌 Points clés à retenir

  • La formule maîtresse : U = R × I
  • Pour trouver I : I = U ÷ R
  • Pour trouver R : R = U ÷ I
  • Si R augmente et U reste fixe → I diminue
  • Si U augmente et R reste fixe → I augmente
```
LEÇON 03

Résistances en Série

Qu'est-ce que la mise en série ?

Des résistances sont en série quand elles sont branchées les unes après les autres sur le même fil. Le courant passe par toutes les résistances dans l'ordre, comme des portes à franchir en file.

```

FORMULE SÉRIE :

R_total = R1 + R2 + R3 + …

Propriétés importantes :

  • → R_total est toujours > à chaque résistance seule
  • → Le même courant I passe partout dans le circuit
  • → La tension U se divise entre les résistances
``` Résistances en SÉRIE : +–[R1]–[R2]–[R3]–+ | 10Ω 20Ω 30Ω | | | [U=60V] | | | +———————+ R_total = 10+20+30 = 60Ω I = U/R = 60/60 = 1A Même I partout = 1A
```

🔧 Exemple complet

Deux résistances R1=15Ω et R2=25Ω en série, alimentées par U=40V.

Étape 1R_total = R1+R2 = 15+25 = 40 Ω
Étape 2I = U÷R_total = 40÷40
RésultatI = 1 A dans tout le circuit
💡
Application militaire : Si une résistance en série tombe en panne (circuit ouvert), TOUT le circuit s'arrête. C'est le principe du guirlande de Noël : une ampoule grillée coupe toutes les autres. En avionique, on évite ce schéma pour les systèmes critiques.

📌 Points clés à retenir

  • Série = résistances bout à bout, même fil
  • R_total = somme de toutes les résistances
  • Le courant I est identique partout dans le circuit
  • La tension totale U se répartit sur chaque résistance
  • Si une résistance lâche → circuit coupé pour tout
```
LEÇON 04

Résistances en Parallèle

Qu'est-ce que la mise en parallèle ?

Des résistances sont en parallèle quand elles sont branchées côte à côte, entre les mêmes deux points. Le courant se divise entre elles, comme des chemins alternatifs.

```

FORMULE PARALLÈLE :

1/R_total = 1/R1 + 1/R2 + …
Cas 2 résistances :
R_total = (R1 × R2) ÷ (R1 + R2)

Propriétés importantes :

  • → R_total est toujours plus petite que la plus petite résistance
  • → La même tension U est aux bornes de chaque branche
  • → Le courant total I est la somme des courants de chaque branche
``` Résistances en PARALLÈLE : ``` +--[R1=6Ω]--+ | | ``` [U=12V]+ + | | +–[R2=12Ω]–+ R_t = (6×12)÷(6+12) = 72÷18 R_total = 4 Ω (< 6Ω !) I_total = 12÷4 = 3A
```

🔧 Exemple détaillé

R1=6Ω et R2=12Ω en parallèle sous U=12V

Étape 1R_t = (6×12)÷(6+12) = 72÷18 = 4 Ω
Étape 2I_total = 12÷4 = 3 A
VérifI1=12/6=2A, I2=12/12=1A → 2+1=3A ✓

ASTUCE RAPIDE :

Deux résistances égales en parallèle → R_total = R÷2
Ex : 20Ω // 20Ω = 10Ω

💡
Application militaire : Les systèmes critiques d'un aéronef (commandes, éclairage de secours) sont câblés en parallèle. Si une branche tombe en panne, les autres continuent de fonctionner. La résistance totale baisse quand on ajoute des branches → plus de courant total → prévoir des fusibles adaptés !

📌 Points clés à retenir

  • Parallèle = résistances sur des branches séparées entre 2 mêmes points
  • R_total < chaque résistance individuelle (toujours !)
  • Même tension U sur chaque branche
  • Formule 2 résistances : R_t = (R1×R2)÷(R1+R2)
  • Si une branche lâche → les autres continuent de fonctionner
```
LEÇON 05

Lecture de Schémas & Sécurité Électrique

Symboles essentiels à reconnaître

SYMBOLES DE BASE : Batterie/Source : ─|├─ (longue = +) Résistance : ─[R]─ ou ─///─ Interrupteur ouvert : ─/ ─ (coupé) Interrupteur fermé : ─/─ (passant) Fusible : ─[F]─ ou ─[≡]─ Moteur : ─(M)─ Lampe : ─(L)─ ou ─⊗─ Masse/GND : ─⏚
```

Règles de lecture d'un schéma :

  1. Repérer la source de tension (batterie)
  2. Suivre le chemin du courant (+) vers (−)
  3. Identifier les interrupteurs : ouverts = circuit coupé
  4. Repérer les fusibles et leur état
  5. Vérifier si le circuit est en série ou parallèle

Situations de panne à reconnaître

🔌 INTERRUPTEUR OUVERT
Circuit coupé → I = 0 A → rien ne fonctionne
Le circuit est "ouvert" = pas de chemin pour l'électricité
🔥 FUSIBLE GRILLÉ
Le fusible a "sacrifié" pour protéger le circuit
Un fusible grillé = circuit coupé jusqu'au remplacement
Cause : courant trop élevé (court-circuit ou surcharge)
⚙️ MOTEUR BLOQUÉ
Un moteur bloqué ne tourne plus → résistance augmente → courant chute
Ou : moteur bloqué → résistance → courant monte = surchauffe
⚡ COURT-CIRCUIT
Résistance quasi nulle → courant très élevé → fusible ou câble brûle
DANGER : risque d'incendie électrique sur l'aéronef
``` Exemple : circuit avec fusible et moteur +––[F]––[SW]––+—–(M)––+ | Fusible Interr. | Moteur | [28V] | | | +—[R1]—+ | | | | | +––GND—––+———–+––+ Si SW ouvert → I=0A, moteur arrêté Si F grillé → circuit coupé, vérifier surcharge Si M bloqué → R monte → courant change
```

📌 Points clés à retenir

  • Interrupteur ouvert = circuit coupé = I = 0 A
  • Fusible grillé = protection contre surintensité = remplacement nécessaire
  • Moteur bloqué mécaniquement → problème électrique en conséquence
  • Court-circuit = résistance ≈ 0 → I très élevé → danger d'incendie
  • Toujours analyser le schéma avant d'intervenir : localiser la source, les interrupteurs, les fusibles
```
